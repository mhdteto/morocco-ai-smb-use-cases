# Roadmap

## Current version

v1.0 — Initial public version of the Morocco AI SMB Use Cases library.

## Purpose

This project provides practical AI use cases for Moroccan SMBs across common business sectors.

The goal is to help business owners, consultants and B2B teams identify realistic AI opportunities, prioritize them by business value and complexity, and start with focused pilot projects.

## Current scope

The current version includes:

- AI use cases by sector.
- Prioritization matrix.
- SMB readiness checklist.
- Pilot selection guide.
- ROI estimation guide.
- Example client scenarios.
- Risk and limitation notes.
- Business-focused documentation.
