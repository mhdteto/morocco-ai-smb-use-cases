# AI Pilot Planning Template

Use this template before launching an AI or automation pilot.

## 1. Pilot name

```txt
Example: AI-assisted lead qualification
```

## 2. Business problem

Describe the problem clearly.

Example:

```txt
The sales team receives inbound leads but spends too much time manually reviewing, qualifying and routing them.
```

## 3. Current workflow

Describe the current manual process.

Example:

```txt
1. Lead arrives from website form.
2. Sales rep checks the message manually.
3. Rep decides whether the lead is qualified.
4. Rep writes a follow-up message.
5. Rep updates the CRM.
```

## 4. Proposed AI workflow

Describe the AI-assisted process.

Example:

```txt
1. Lead arrives from website form.
2. AI classifies the lead intent.
3. AI suggests priority level.
4. AI drafts a follow-up message.
5. Human reviews the result.
6. CRM is updated after approval.
```

## 5. Inputs required

List the required inputs.

Examples:

- Lead name.
- Company name.
- Email.
- Message.
- Industry.
- Budget, if available.
- Service requested.
- Source channel.

## 6. Expected output

List the expected outputs.

Examples:

- Lead category.
- Priority level.
- Suggested next action.
- Follow-up draft.
- CRM notes.
- Owner recommendation.

## 7. Human review

Define who reviews the AI output.

```txt
Sales owner reviews before sending any external message.
```

## 8. Success metrics

Choose measurable indicators.

Examples:

- Time saved per lead.
- Response time.
- Lead qualification consistency.
- Follow-up quality.
- CRM note completeness.
- Sales rep adoption.

## 9. Pilot duration

Recommended:

```txt
2 to 4 weeks
```

## 10. Risks

List possible risks.

Examples:

- AI misclassifies leads.
- Output sounds generic.
- CRM data is incomplete.
- Team does not adopt the workflow.
- Sensitive data is used incorrectly.

## 11. Decision after pilot

At the end of the pilot, decide:

- keep;
- improve;
- stop;
- scale.
