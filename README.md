# Morocco AI SMB Use Cases

Practical AI use cases for Moroccan SMBs by sector, complexity and business value.

## Why it matters

AI adoption for SMBs works best when use cases are practical, close to business operations and easy to prioritize. This repository maps common opportunities for Moroccan businesses by sector.

It is a practical AI use case library for Moroccan SMBs that want to identify realistic, high-value AI and automation opportunities.

## What is included

- AI use cases by sector
- Prioritization matrix
- SMB AI readiness checklist
- AI pilot planning template
- ROI estimation guide
- Risk and limitation notes
- Example client scenarios
- Roadmap
- Changelog

## How to use

1. Choose the sector closest to your business.
2. Review the relevant use cases.
3. Compare value, complexity, data readiness and adoption readiness in the [prioritization matrix](prioritization-matrix.md).
4. Select one narrow, low-risk pilot.
5. Measure the result before scaling.

## Example

A real estate agency can start with lead qualification, property description drafting or WhatsApp FAQ support before investing in more advanced automation.

## Business value

The goal is to help SMB leaders move from generic AI interest to concrete workflows that may save time, improve consistency and support better customer experiences.
