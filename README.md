# Morocco AI SMB Use Cases

Practical AI use cases for Moroccan SMBs by sector, complexity and ROI potential.

## Why it matters

AI adoption for SMBs works best when use cases are practical, close to business operations and easy to prioritize. This repository maps common opportunities for Moroccan businesses by sector.

## What is included

- Real estate use cases
- Retail use cases
- Education use cases
- B2B sales use cases
- Hospitality use cases
- Professional services use cases
- Prioritization matrix

## Use case

Use this repository to identify where AI can create practical value for Moroccan SMBs without starting with complex machine learning projects.

## Example

A real estate agency can start with lead qualification, property description generation and WhatsApp FAQ support before investing in more advanced automation.

## How to use

1. Choose the sector closest to your business.
2. Review the use cases.
3. Compare complexity and ROI potential in `prioritization-matrix.md`.
4. Start with one low-risk workflow.
5. Measure time saved, lead quality or customer response speed.

## Business value

The goal is to help SMB leaders move from generic AI interest to concrete workflows that save time, improve sales and support better customer experiences.
